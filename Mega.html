  • str.lower() ‚Äì –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –≤—Å–µ —Å–∏–º–≤–æ–ª—ã –≤ –Ω–∏–∂–Ω–∏–π —Ä–µ–≥–∏—Å—Ç—Ä
  • str.strip() ‚Äì —É–¥–∞–ª—è–µ—Ç –ø—Ä–æ–±–µ–ª—ã/–ø–µ—Ä–µ–≤–æ–¥—ã —Å—Ç—Ä–æ–∫ –ø–æ –∫—Ä–∞—è–º —Å—Ç—Ä–æ–∫–∏

  • str.rstrip() ‚Äì —É–¥–∞–ª—è–µ—Ç –ø—Ä–æ–±–µ–ª—ã —Å–ø—Ä–∞–≤–∞
  • str.replace(old, new) ‚Äì –∑–∞–º–µ–Ω—è–µ—Ç –ø–æ–¥—Å—Ç—Ä–æ–∫—É –Ω–∞ –¥—Ä—É–≥—É—é
  • str.split(separator) ‚Äì —Ä–∞–∑–±–∏–≤–∞–µ—Ç —Å—Ç—Ä–æ–∫—É –Ω–∞ —Å–ø–∏—Å–æ–∫ –ø–æ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—é
  • str.join(iterable) ‚Äì –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã —Å–ø–∏—Å–∫–∞ –≤ —Å—Ç—Ä–æ–∫—É —á–µ—Ä–µ–∑ —Å—Ç—Ä–æ–∫—É-—Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å
  • str.find(substring) ‚Äì –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ –≤—Ö–æ–∂–¥–µ–Ω–∏—è –ø–æ–¥—Å—Ç—Ä–æ–∫–∏ –∏–ª–∏ -1
  • str.index(substring) ‚Äì –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ find(), –Ω–æ –≤—ã–∑—ã–≤–∞–µ—Ç –æ—à–∏–±–∫—É, –µ—Å–ª–∏ –Ω–µ –Ω–∞–π–¥–µ–Ω–æ
  • str.startswith(prefix) ‚Äì –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –ª–∏ —Å—Ç—Ä–æ–∫–∞ —Å —É–∫–∞–∑–∞–Ω–Ω–æ–π –ø–æ–¥—Å—Ç—Ä–æ–∫–∏
  • str.endswith(suffix) ‚Äì –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è –ª–∏ —Å—Ç—Ä–æ–∫–∞ —É–∫–∞–∑–∞–Ω–Ω–æ–π –ø–æ–¥—Å—Ç—Ä–æ–∫–æ–π
  • str.isdigit() ‚Äì –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç True, –µ—Å–ª–∏ –≤—Å—è —Å—Ç—Ä–æ–∫–∞ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Ü–∏—Ñ—Ä
  • str.isalpha() ‚Äì –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç True, –µ—Å–ª–∏ –≤—Å—è —Å—Ç—Ä–æ–∫–∞ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ –±—É–∫–≤
  • str.isalnum() ‚Äì –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç True, –µ—Å–ª–∏ –≤—Å—è —Å—Ç—Ä–æ–∫–∞ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ –±—É–∫–≤ –∏ —Ü–∏—Ñ—Ä
  • str.count(substring) ‚Äì —Å—á–∏—Ç–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π –ø–æ–¥—Å—Ç—Ä–æ–∫–∏
  • str.title() ‚Äì –¥–µ–ª–∞–µ—Ç –ø–µ—Ä–≤—É—é –±—É–∫–≤—É –∫–∞–∂–¥–æ–≥–æ —Å–ª–æ–≤–∞ –∑–∞–≥–ª–∞–≤–Ω–æ–π
  • str.capitalize() ‚Äì –¥–µ–ª–∞–µ—Ç –ø–µ—Ä–≤—É—é –±—É–∫–≤—É —Å—Ç—Ä–æ–∫–∏ –∑–∞–≥–ª–∞–≤–Ω–æ–π, –æ—Å—Ç–∞–ª—å–Ω—ã–µ —Å—Ç—Ä–æ—á–Ω—ã–µ
  • str.swapcase() ‚Äì –º–µ–Ω—è–µ—Ç —Ä–µ–≥–∏—Å—Ç—Ä –≤—Å–µ—Ö —Å–∏–º–≤–æ–ª–æ–≤ –Ω–∞ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω—ã–π

üî¢ –ß–∏—Å–ª–∞ (int, float, complex)

  • float ‚Äì –¥—Ä–æ–±–Ω—ã–µ —á–∏—Å–ª–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä: 3.14)
  • complex ‚Äì –∫–æ–º–ø–ª–µ–∫—Å–Ω—ã–µ —á–∏—Å–ª–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä: 2+3j)
  • + ‚Äì —Å–ª–æ–∂–µ–Ω–∏–µ
  • - ‚Äì –≤—ã—á–∏—Ç–∞–Ω–∏–µ
  • * ‚Äì —É–º–Ω–æ–∂–µ–Ω–∏–µ
  • / ‚Äì –¥–µ–ª–µ–Ω–∏–µ (–≤—Å–µ–≥–¥–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç float)
  • // ‚Äì —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–µ –¥–µ–ª–µ–Ω–∏–µ
  • % ‚Äì –æ—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è
  • ** ‚Äì –≤–æ–∑–≤–µ–¥–µ–Ω–∏–µ –≤ —Å—Ç–µ–ø–µ–Ω—å
  • abs(x) ‚Äì –º–æ–¥—É–ª—å —á–∏—Å–ª–∞
  • round(x, n) ‚Äì –æ–∫—Ä—É–≥–ª–µ–Ω–∏–µ –¥–æ n –∑–Ω–∞–∫–æ–≤ –ø–æ—Å–ª–µ –∑–∞–ø—è—Ç–æ–π
  • pow(a, b) ‚Äì –≤–æ–∑–≤–µ–¥–µ–Ω–∏–µ –≤ —Å—Ç–µ–ø–µ–Ω—å
  • divmod(a, b) ‚Äì –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–æ—Ä—Ç–µ–∂ (—á–∞—Å—Ç–Ω–æ–µ, –æ—Å—Ç–∞—Ç–æ–∫)
